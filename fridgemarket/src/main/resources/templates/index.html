아래 버튼을 눌러 소셜 로그인을 시작하세요.

Google로 로그인

참고 사항:

1. 이 페이지는 `/` (루트) 경로로 접근할 때 보입니다.

2. Google OAuth2 클라이언트 ID와 Secret이 `application.properties`에 올바르게 설정되어야 합니다.

3. Google Cloud Console에서 승인된 리디렉션 URI에 `http://localhost:8080/login/oauth2/code/google`을 추가해야 합니다.

4. 로그인 성공 시 `/success` 페이지로, 실패 시 `/loginFailure` 페이지로 이동합니다.
